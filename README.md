Birthday Website - Priyanshi

Overview
- Premium dark birthday landing page with neon blue, purple, and gold accents.
- Includes hero intro, animated gallery, scroll timeline, typed message, surprise fireworks/confetti, synth-based music toggle, and glowing ending.

Run
- Open `index.html` in a browser. No build step is required.

Notes
- Gallery images currently use remote placeholder photography so the layout looks complete immediately.
- Swap the image URLs in `index.html` with real photos whenever you are ready.
- Background music is generated in the browser with the Web Audio API, so no local audio file is required.

Files
- `index.html`: page structure and content
- `styles.css`: layout, responsive design, glow effects, and animation styling
- `script.js`: particles, lightbox, scroll reveals, typing effect, celebration effects, and music engine
