Birthday Website - Priyanshi

Overview
- Warm cinematic single-page birthday website built as a polished upgrade from the backup version.
- Uses a deeper romantic palette with rose-gold, champagne, blush, and warm plum tones.
- Includes animated ambient background, hero spotlight, scroll reveals, typed message, gallery lightbox, surprise celebration burst, and local image assets.

Run
- Open `index.html` in a browser. No build step is needed.

Files
- `index.html`: structure and content
- `styles.css`: theme, layout, responsive design, and animation styling
- `script.js`: ambient canvas, lightbox, reveals, typing effect, petals, and celebration effects
- `images/`: local gallery images

Notes
- The writing has been kept simpler and more human on purpose.
- Gallery photos are local files, so the page works without remote image URLs.
